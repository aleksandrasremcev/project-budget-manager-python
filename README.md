# Project Budget Manager

A Python application for managing project budgets. The application allows users to create, update, search, delete, and store project budget information using CSV files.

---

## Features

- Add new projects
- View all projects
- Search projects by name
- Update project information
- Delete projects
- Save project data to CSV
- Load project data from CSV
- Calculate:
  - Total approved budget
  - Total actual costs
  - Remaining budget
  - Budget status (within budget / exceeded)

## Installation

Clone the repository:

```bash
git clone https://github.com/aleksandrasremcev/project-budget-manager-python.git
```

Go to the project folder:

```bash
cd project-budget-manager-python
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
